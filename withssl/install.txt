	这个文件介绍如何安装IoT Edge SDK, 以及它所依赖的第三方库，并且支持SSL连接。
	下面的步骤以ubuntu为例，不同版本的操作系统对应的命令可能有细微的差别。
	
	安装步骤：
	1,安装git:
		sudo apt-get install git

	2,安装autoconf:
		sudo apt-get install autoconf

	3,安装libtool:
		sudo apt-get install libtool

	4,安装libssl：
		sudo apt-get install libssl-dev

	(下面的步骤假设本sdk是下载在目的 ~/iot-edge-sdk-for-iot-parser，如果你的目录不是这里，请把下面涉及到的地方都一一更正）
	5,创建一个临时目录
		mkdir ~/deps
	5,下载并安装cJSON:
		cd ~/deps
		git clone https://github.com/DaveGamble/cJSON.git
		cd cJSON && sudo make install

	6,下载并安装libmodbus:
		cd ~/deps
		git clone https://github.com/stephane/libmodbus.git
		cd libmodbus && ./autogen.sh && ./configure && sudo make install

	7,下载并安装paho.mqtt.c:
		cd ~/deps
		git clone https://github.com/eclipse/paho.mqtt.c.git
		cp paho.mqtt.c/src/VersionInfo.h.in paho.mqtt.c/src/VersionInfo.h
		cp ~/iot-edge-sdk-for-iot-parser/src/withssl/paho_Makefile_cs paho.mqtt.c/Makefile
		cd paho.mqtt.c && sudo make install 
	
	8,使前面安装的lib生效：
		sudo ldconfig

	9,编译IoT Edge SDK:
		~/iot-edge-sdk-for-iot-parser/withssl
		make

	至此，网关程序bdModbusGateway应该生成在父目录了。可以直接./bdModbusGateway运行了。
